# BLE Player

VORZE U.F.O. SA をブラウザから制御するプレイヤーアプリ。  
動画・音声ファイルの再生に合わせて、CSVで記述したタイミングでデバイスを動作させます。

---

## 動作環境

| 環境 | 対応状況 |
|------|----------|
| PC（Chrome / Edge） | ✅ フル対応 |
| Android（Chrome） | ✅ 対応（ファイル操作除く） |
| iOS | ❌ Web Bluetooth API 非対応 |

**HTTPS 必須です。** ローカルの `file://` では Web Bluetooth API が動作しません。  
GitHub Pages などの HTTPS 環境で使用してください。

## フォルダ構成

フォルダをそのままアプリに読み込ませます。構成は以下を想定しています。

```
作品フォルダ/
├── track01.mp4       # 動画または音声ファイル
├── track01.csv       # タイミングデータ（省略可）
├── cover.jpg         # サムネイル（音声再生時に表示）
└── subFolder/        # サブフォルダ（任意の深さ）
      ├── track01.mp4
      └── track01.csv
```

### CSVとトラックの紐づけ

| 状況 | 動作 |
|------|------|
| CSVが0個 | 全トラック未割当 |
| CSVが1個 | 全トラックに同じCSVを適用 |
| CSV数 ＝ トラック数 | ファイル名昇順で順番に紐づけ |
| それ以外 | ファイル名が一致するもののみ紐づけ |

---

## CSVフォーマット

```csv
time_ms,direction,strength
0,0,0
100,1,50
200,0,80
```

| カラム | 内容 |
|--------|------|
| `time_ms` | 再生開始からの経過時間（**デシ秒** = 1/10秒） |
| `direction` | 向き（`0` = 右、`1` = 左） |
| `strength` | 強さ（`0` 〜 `100`） |

## 使い方

### 1. 接続

アプリを開くと BLE 接続画面が表示されます。

1. **「UFO SA に接続する」** をタップ
2. ブラウザのデバイス選択ダイアログで **UFOSA** を選択
3. 接続成功後、メイン画面へ遷移

動作確認したい場合は **「接続せずに続ける（テスト用）」** でスキップできます。

### 2. フォルダを追加

- **PC**：メイン画面に遷移すると自動でフォルダ選択ダイアログが開きます
- **Android**：サイドバー下部の **「＋ フォルダを追加」** をタップ

初回追加時はメタデータ（作品名・サークル名・声優名・RJ番号）の入力画面が表示されます。スキップ可能です。

### 3. 再生

サイドバーから作品を選択 → トラックをタップで再生が始まります。  
CSVが紐づいていれば、再生位置に合わせてデバイスが自動で動作します。

### 4. CSV編集

トラックリスト右上の **「CSV編集」** から CSV 編集画面を開けます。

- タイムラインをタップしてマーカーを追加
- ミラーリング・繰り返し・フェードイン/アウトなどのツールが使えます
- **PC**：編集したCSVをファイルに直接上書き保存
- **Android**：ダウンロードして手動で差し替え

## BLE仕様

| 項目 | 内容 |
|------|------|
| デバイス名 | `UFOSA` |
| Service UUID | `40ee1111-63ec-4b7f-8ce7-712efd55b90e` |
| Characteristic UUID | `40ee2222-63ec-4b7f-8ce7-712efd55b90e` |
| コマンド形式 | `[0x02, 0x01, byte2]` |
| byte2 | MSB = 向き（0:右 / 1:左）、下位7bit = 速度（0〜100） |

---

## ライブラリ管理

- 作品のメタデータは **localStorage** に保存されます（ブラウザを閉じても保持）
- サイドバー下部からライブラリを **JSON でエクスポート・インポート** できます
- 機種変更やブラウザ移行の際はエクスポートしておくことを推奨します

## 技術スタック

| 技術 | 用途 |
|------|------|
| Web Bluetooth API | UFO SA との BLE 通信 |
| File System Access API | PC でのファイル読み書き |
| IndexedDB | CSV 編集画面へのファイル受け渡し |
| localStorage | ライブラリ・パターンの永続化 |
| Vanilla JS / HTML / CSS | フレームワーク不使用 |
